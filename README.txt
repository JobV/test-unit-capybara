= Test::Unit::Capybara

* http://rubyforge.org/projects/test-unit/

== DESCRIPTION:

test-unit-capybara - Capybara adapter for Test::Unit.

== FEATURES/PROBLEMS:

* This provides Capybara integrated Test::Unit::TestCase.

== INSTALL:

* sudo gem install test-unit-capybara

== USAGE:

  require 'test/unit/capybara'
